# ScholarScout
An AI agent for academic professionals to find the next destination for their higher studies.
(Work in progress)

# Setup
```
python3 -m pip install -r requirements.txt
```

# Code overview
The data scraping scripts exists in scripts/scraper. Use main.py for scraping the catalogue.
